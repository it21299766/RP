## üéØ Problem Statement

Universities worldwide face significant operational inefficiencies:

- **Class & Exam Scheduling:** Rigid systems leading to suboptimal room utilization and timetable clashes
- **Staff Workload Distribution:** Imbalanced assignments resulting in overburdened staff and reduced productivity
- **Laboratory/Resource Utilization:** Under-utilized or over-subscribed facilities due to poor management
- **Transport Management:** Inefficient shuttle routes and vehicle usage, increasing costs and environmental impact

**Impact:** Wasted resources, overburdened staff, and dissatisfied students.

## üí° Proposed Solution

### 1. Adaptive Task & Responsibility Allocation System
**Developer:** Gunarathne N.R. (IT22892508)

An AI-powered system that ensures fair and intelligent distribution of staff duties using:
- **Integer Linear Programming (ILP)** for optimization
- **Genetic Algorithms (GA)** for flexible allocation
- **Reinforcement Learning (RL)** for adaptive learning
- Interactive chatbot interface for seamless interaction

**Key Technologies:**
- Google OR-Tools, Gurobi
- Pandas, Matplotlib, Plotly
- Rule engine for policy compliance

### 2. Hybrid Scheduling Framework
**Developer:** Aloka M.D.C. (IT22138354)

A resilient timetable management system combining:
- **ILP/MIP** for globally optimized semester planning
- **Tabu Search/Simulated Annealing** for real-time disruption handling
- Nightly healing process to optimize accumulated changes
- Complete audit logging for transparency

**Key Technologies:**
- Gurobi, CPLEX, OR-Tools
- SQL/MongoDB for data storage
- Metaheuristics for adaptive reallocation

### 3. Dynamic Timetable Generator
**Developer:** Wickramarathna W.G.T.A. (IT21339424)

Genetic Algorithm-based system for scalable, conflict-free timetable generation:
- Real-time adaptability to faculty unavailability
- Smart resource release and cancellation handling
- Role-based access (Admin, Teacher, Student)

**Key Technologies:**
- Java & JSP
- React, Spring Boot
- MySQL Database
- Apache Tomcat

### 4. Smart Shuttle & Transport Route Optimizer
**Developer:** Dharmarathna D.A.T.N. (IT21299766)

Dynamic route optimization system using:
- **K-Means Clustering** for geographical student grouping
- **Genetic Algorithm** for route optimization
- Real-time route adjustment based on student registrations

**Key Technologies:**
- Python (scikit-learn, NumPy)
- Spring Boot + React
- Google Maps API / Leaflet
- Vehicle Routing Problem (VRP) formulation
